# Demo WebApp - Portal de Procesos Knauf

Esta es una **demo conceptual** de una webapp muy sencilla para visualizar procesos y documentos
desde un navegador, incluyendo dispositivos móviles.

## Características

- Vista de lista de procesos (ejemplo: Calcinación, Mezclado, Despacho).
- Cada proceso muestra sus documentos asociados.
- El usuario puede:
  - Buscar por nombre de proceso, área o tipo de documento.
  - Abrir un documento en un modal y ver una descripción básica.
- Maquetado responsive, usable desde un smartphone.

## Cómo usar

1. Descarga el ZIP y descomprímelo.
2. Abre el archivo `index.html` en tu navegador.
3. Prueba la búsqueda y la apertura de documentos.

En un proyecto real, esta misma estructura podría conectarse a un backend
(SharePoint, API interna, base de datos, etc.) para mostrar los documentos reales.
